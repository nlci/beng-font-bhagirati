bhagirati_unicode.xml - mapping from legacy to Unicode as used by ttfbuilder
BEN Bhagirati 2 Unicode CONV chart.ods - source for bhagirati_unicode.xml
BEN Bhagirati 2 Unicode CONV chart.xls - source for the .ods version, can be saved from the .ods version for use on Windows
