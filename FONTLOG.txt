FONTLOG
Bhagirati - NLCI Bengali fonts
==============================

This file provides detailed information on the NLCI Bengali fonts.
This information should be distributed along with the NLCI Bengali fonts and any derivative works.

Basic Font Information
----------------------

Pan Bengali font designed to support all the languages using the Bengali script.
